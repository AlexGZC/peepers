Peepers client

This is a minimal example of what is needed for a peepers client (well MJPEG over HTTP client).

Instructions

  1. You will need to be on the same network as Peepers, e.g., your home wifi.
  2. Start Peepers first and then enter the address it displays into the text box below.

This has been tested on Chromium Version 24.0.1312.70 (181759) and Firefox 18.0.2.

If this works you may be interested in going full screen.
